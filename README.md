A simple and responsive web-based Age Calculator that calculates a user's exact age in years, months, and days using JavaScript.

## 🚀 Features
- User-friendly interface
- Input Date of Birth
- Calculates accurate age (Years, Months, Days)
- Input validation (empty & future date)
- Responsive design

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript (Date Object)

## 📚 Key Concepts Used
- DOM Manipulation
- JavaScript Date & Time
- Input Validation
- Event Handling

## 📂 Project Structure
age-calculator/
│── index.html
│── style.css
│── script.js


## ▶️ How to Run
1. Download or clone the repository
2. Open `index.html` in your browser
3. Enter your Date of Birth
4. Click on "Calculate Age"
